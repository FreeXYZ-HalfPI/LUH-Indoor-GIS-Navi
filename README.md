# LUH-Indoor-GIS-Navi
A Free Software prototype of an indoor navigation system for Leibniz University Hannover using GIS, graph-based routing and building floor plans.

# Project Overview
Finding rooms inside university buildings is often difficult for new students and visitors. Outdoor navigation ends at the building entrance, leaving users without guidance inside complex buildings.

# Goal
Develop an open-source indoor GIS navigation system capable of routing users from outdoor GPS navigation to indoor destinations using floorplan-based graph routing.

# Features (future)
- Indoor routing

- Multi-floor buildings

- Outdoor → Indoor transition

- QR-code positioning

- Accessible routing

- Mobile interface

- Digital Twin integration
